# Common Automation Tasks

Use these examples as templates for your automation requests.

## 1. Basic Smoke Test
"Go to http://localhost:5173, click 'New Chat', type 'Hello', and wait for a reply."

## 2. Verify Settings
"Go to http://localhost:5173/settings, change the theme to 'Dark', and click save."

## 3. Test Model Switching
"Go to http://localhost:5173, open the model selector, select 'Llama-3', and send a message."

## 4. Check for Errors
"Go to http://localhost:5173, send a long message, and take a screenshot if an error toast appears."

## 6. Coordinate-Based Task
"Go to https://news.ycombinator.com, find the 'new' link, and click it using coordinates."

## 7. Iterative Verification
"Go to https://news.ycombinator.com, click 'new', and then iteratively find and print the text of the first 3 stories."

## 8. Hybrid Visual/DOM Task
"Go to the homepage, find the search icon at roughly (950, 50), click it, and type 'Gemini'."
